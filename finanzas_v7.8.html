Configuración IA — Claude API

La interpretación IA funciona a través de tu Google Apps Script (evita restricciones del navegador). La API key de Claude se guarda en Script Properties del proyecto Apps Script, no en el browser.

Pasos para configurar:
1. Abrí tu proyecto en script.google.com
2. Hacé click en el engranaje ⚙️ → Propiedades de secuencia de comandos
3. Agregá una propiedad: CLAUDE_API_KEY → tu key de Anthropic
4. Guardá y volvé a deployar el Apps Script como Web App

🔑 Obtené tu key: console.anthropic.com/keys · Costo típico por extracto: < U$D 0.01
